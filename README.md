# Aruvili UI

Aruvili UI is an enterprise-grade React component library. Designed for high-fidelity interfaces, it provides a unified aesthetic tailored for modern, professional web applications.

---

## Overview

Aruvili UI is built on a Turborepo monorepo architecture. Our core philosophy emphasizes pristine design, strict type safety, and a seamless developer experience without relying on complex styling overrides.

### Features
- **React Focus**: First-class components built exclusively for React.
- **Enterprise Aesthetic**: Deep dark-mode default, meticulously crafted borders, and high-contrast typography.
- **Scalable Architecture**: Managed via `pnpm` workspaces and Turborepo.

---

## Workspace Structure

- `apps/playground`: A Vite-powered React application demonstrating the interactive components.
- `packages/react`: The core React component library.
- `packages/specs`: Centralized TypeScript interfaces for component APIs.
- `packages/core`: Shared Tailwind CSS configuration and design tokens.

---

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Start the local development server:

```bash
pnpm dev
```
